** multiple cursor

|----------------+------------------|

| =gzD= or =M-D= | 生成鼠标-上      |
| =gzj=          | 生成鼠标-移动下  |

| =gzq=          | undo all         |
| =gzt=          | toggle cursor    |
| =gzz=          | make cursor here |
| =gzI=          | unknow           |
| =gzA=          | unknow           |
